# 概念→视觉隐喻指南

抽象概念必须翻译为图像生成模型可理解的视觉元素。以下为常见领域的隐喻映射参考：

| 抽象概念 | 视觉隐喻 |
|---------|---------|
| 分布式/一致性 | 一组互连节点、投票箱、绿色确认标识、环形拓扑 |
| 分区/隔离 | 玻璃屏障、虚线隔离框、不同颜色区域、水区 |
| 加密/安全 | 半透明盾牌层、锁/钥匙符号、二进制流、发光轮廓 |
| 延迟/性能 | 沙漏/秒表、速度线、赛车道、流量密度渐变 |
| 数据流/管道 | 发光粒子管、河流分支、管道截面、波浪带 |
| 微服务/模块化 | 船运集装箱、乐高块、独立舱室、连接桥 |
| 可观测性/监控 | 仪表盘、波形图、雷达图、信号灯 |
| CI/CD/部署 | 流水线、传送带、绿灯/红灯状态、登台区 |
| 缓存/存储 | 堆叠层、蜂巢格、书架、存储单元阵列 |
| 机器学习/训练 | 神经网络节点、数据集方块、训练→推理路径 |

**使用规则**：优先匹配源文中出现的术语，选择最贴合的隐喻。避免强行套用——如果源文有自带的比喻（如 "think of it as a highway" ），优先使用原文比喻而非表内映射。
