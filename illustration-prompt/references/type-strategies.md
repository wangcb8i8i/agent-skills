# Type-specific Prompt Strategies

按配图类型选择对应的 Prompt 策略。

## 概念示意图

| 维度 | 策略 |
|------|------|
| SUBJECT | 侧重抽象→具象的视觉翻译。使用拓扑/网络/结构来表示概念关系。主体元素必须包含原文中的重要术语和名词实体。 |
| COMPOSITION | 推荐径向/网络/节点-边布局。以结构清晰度为第一优先级。省略非必要纹理细节。 |
| COLOR | 用不同色相/饱和度标明组件状态差异（如活跃态 vs 待机态）。连接线颜色区分关系类型。 |
| LIGHTING | 均匀环境光为主，避免戏剧化阴影。允许自发光元素突出关键节点。 |
| ANNOTATIONS | 标注连接关系的方向与类型。如有节点状态变化，加注状态标签或色阶指示器。 |
| VIEWPOINT | 推荐等轴测 3/4 视图或正视图，确保所有节点可视。 |
| FOCUS | 中心焦点或分层焦点，核心概念节点做视觉强调。 |
| SEPARABILITY | 节点间距均匀，标签在节点外侧，连接线不穿过节点。 |

## 架构图

| 维度 | 策略 |
|------|------|
| SUBJECT | 侧重组件分层、边界界定、连接线方向。使用容器/盒子/层级布局。每层用半透明背景色区分。 |
| COMPOSITION | 水平分层（L→R）或垂直分层（T→B）。边界清晰，容器嵌套关系明确。 |
| COLOR | 每层不同色相，层间颜色过渡使用 HSL 渐变。边框半透明度区分层级深度。 |
| LIGHTING | 扁平光照为主，层内用微弱渐变增强深度感。禁止强对比光导致的边界模糊。 |
| MOTION | 连接线标注方向和数据/请求类型（实线/虚线/点线）。线宽与流量/优先级成正比。 |
| ANNOTATIONS | **必选**。保留组件标签文字，层名左上角标注。图例标注线型和颜色含义。 |
| VIEWPOINT | 推荐正面平视图或 30° 等轴测。禁止极端角度导致重要组件被遮挡。 |
| SEPARABILITY | 层级边界清晰，容器内元素不溢出，连接线沿层级边缘走线。 |
| EXCLUSIONS | 除非标签需要，否则不添加装饰性元素。 |

## 流程图

| 维度 | 策略 |
|------|------|
| SUBJECT | 侧重时序和状态转换。步骤节点 + 决策节点 + 起止节点明确区分。 |
| COMPOSITION | 推荐 L→R 或 T→B 时序布局。使用泳道/分区表示职责归属。多分支路径并行排列。 |
| COLOR | 步骤节点用中性色，决策节点用强调色（如 amber）。泳道用不同低饱和底色区分。状态切换用色阶过渡。 |
| MOTION | **必选**。箭头方向明确（实线为主流程，虚线为分支/异常）。路径交叉处加跳线弧。 |
| ANNOTATIONS | 步骤编号（1/2/3）标注在节点右上角。决策点用菱形标注条件文字。 |
| VIEWPOINT | 正面平视，保持所有节点清晰可读。禁止透视畸变。 |
| SEPARABILITY | 节点间距统一，箭头不交叉穿过节点，决策节点有独立空间。 |
| SCALE & DEPTH | 2 层：前景流程元素 + 背景泳道分区。避免 3+ 层以免干扰阅读顺序。 |

## 信息图

| 维度 | 策略 |
|------|------|
| SUBJECT | 侧重数据对比和视觉层级。使用图表（柱状/饼图/雷达图/气泡图）和比例关系。数值必须精确对应原文数据。 |
| COMPOSITION | 推荐从上到下渐进披露：标题→概览数据→分类细节→图例。对齐方式统一（左对齐或居中）。 |
| COLOR | 不同分类维度用不同色相区分。同一类别内用饱和度渐变表示量级。色盲友好色板优先。 |
| ANNOTATIONS | **必选**。数值标签标注在数据点旁。添加图例或标尺元素。字号层级（标题/类别/数值）至少三级。 |
| VIEWPOINT | 正面直视图，无透视。数据平面平行于画面。 |
| FOCUS | 主数据点居上或居左，按视觉权重从高到低排列。引导元素（颜色高亮/粗边框）将视线引向关键指标。 |
| SEPARABILITY | 数据标签在数据点旁不重叠，图例与主图分离，数值不遮挡图形。 |
| EXCLUSIONS | no decorative elements, no gradients that obscure data readability |

## 装饰配图

| 维度 | 策略 |
|------|------|
| SUBJECT | 氛围优先。使用隐喻而非直白展示。聚焦整体氛围而非具体元素。 |
| COMPOSITION | 松散构图，允许不对称。30% 以上负空间留白。重心不在画面中心。 |
| COLOR | 整图主色调统一，不使用对比色冲突。饱和度≤70%，明度在中高区域。 |
| LIGHTING | 柔和漫射光或单边戏剧光。光晕和色晕允许。 |
| PROPORTIONS | 负空间 ≥30%，主题元素集中在一侧或一角。 |
| TEXTURE | 鼓励纹理叠加（水彩/噪点/纸纹）。渲染风格可偏离矢量扁平。 |
| DETAILS | 无需技术标注或箭头。不需要元素边界清晰。省略非必要的精细结构。 |
| EXCLUSIONS | no text labels, no arrows, no technical annotations, no UI elements |
