# method.md 翻新备忘录

## 什么时候翻新

由你决定。建议的场景：

- 框架级变化出现时（如张坤2025年"基础回报+看涨期权"新框架）
- 收集了大量新语料，直觉上旧提炼可能不够了
- 每半年到一年想做一次例行审视

## 翻新流程

### 第一步：通读语料

阅读顺序（类别比时间重要）：

```
手记/ → 访谈/ → 报告/ （每类内按时间倒序，最新的先读）
```

先读最近的访谈和报告，建立该经理当前框架的印象，再读早期的验证连贯性或找转折点。

### 第二步：逐模块比对

对照 `method.md` 的 7 个模块，逐一问：

| 模块 | 问什么 |
|------|--------|
| 一句话框架 | 还能一句话概括吗？有没有更精确的说法？ |
| 哲学基础 | 核心理念有无偏移？ |
| 选股维度 | 原有维度是否仍代表他的框架？有新增/消失的维度吗？ |
| 特有框架 | 有没有新的分析工具（如DCF四阶段、三种投资情形等）？ |
| 买点/卖点 | 操作纪律有变化吗？ |
| 组合管理 | 集中度/仓位/行业数量有趋势性变化吗？ |
| 风险观 | 对风险的理解有变化吗？ |

### 第三步：更新引用

- 保留旧原文（标注来源），如有更晚、更精炼的表述则增补
- 不要为了"更新而更新"删掉旧引述——它们记录了框架的演变轨迹

### 第四步：更新附录

语料来源对照表增补新文件，标注每个新文件与方法的核心关联。

## 原则

- method.md 是"时点蒸馏物"，不是实时镜像
- 没有自动更新，没有定期提醒
- 翻新时改的是**表述精准度**，不是**框架正确性**——框架是否还"对"由你判断
