# 降级策略参考

信息不足时，按以下顺序执行降级。核心目标是诚实呈现，而非假装确定。

## 降级四步

1. **拓宽搜索** — 放宽时间/地域约束 → 扩展同义词 → 上探一级话题 → 切换信源类型（学术/行业/社区） → 切换搜索语言 → 类比外推（标注为「类比推断，非直接证据」）
2. **评估质量** — 四维可信度评估（权威性/可验证性/时效性/偏见） → 多个低质源交叉一致可提升为中等置信度（三角定位）
3. **处理矛盾** — 定位矛盾类型（数据/方向/归因） → 来源可信度加权对比 → 检查时间差 → 无法裁决则保留双方立场，降级置信度
4. **标注缺口** — 报告标注「信息缺口：{具体缺什么}」，跳过该维度的结论，不强行给出判断

## 四维可信度评估

每个被评为 ★★★ 或 ★★ 的材料执行：

- **来源权威性** — 一手 vs 二手，机构声誉，作者资历
- **数据可验证性** — 数据来源明确？可交叉验证？
- **时效性** — 发布时间，信息是否已过时
- **潜在偏见** — 作者/机构立场，利益冲突，确认偏误倾向

## 决策门

如果核心决策维度经降级后置信度仍 < 50%：

- 输出「针对你的问题，现有信息不足以形成可行动结论」
- 附上什么信息能翻盘、什么渠道可能找到、找到后预期如何影响决策
- 建议用户选择：继续等 / 换角度 / 基于现有信息但接受更高不确定性
