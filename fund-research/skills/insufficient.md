# 数据不足 — 处理流程

**触发条件**：基金成立不到 1 年（`nav_days < 252`），或关键因子 3 窗口全部 N/A。

## 处理规则

1. 不进入取证/合成/判决流程
2. 不输出买/卖/持有等结论
3. 不输出可信度和预期收益

## 输出格式

```
结论：数据不足以做出判断
可信度：N/A
预期收益：N/A

已有数据（截至 {日期}）：
{数据获取状态表}
{已有但不可靠的因子值，标注数据缺口}

注意事项：
- 成立不足 1 年，各时间窗口（1y/3y/5y）均无统计意义
- 同类排名百分位不可用
- 经理任职期 = 基金成立日至今，不能代表历史业绩

建议：
{建议继续持有/观察，N 个月后再评估；或参考经理同类基金历史}
```

## 例外

如果新基金经理有管理同类老基金的历史，可补充搜索经理历史数据作为参考，但必须标注"此结论基于经理历史而非本品基金数据，仅供参考"。
