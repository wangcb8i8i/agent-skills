# 报告模板（参考）

> 这是格式参考，不是封闭 schema。内容以 SKILL.md 输出阶段的 checklist 为准。
> 如果你发现了 checklist 之外但对决策重要的信息，直接新增 section。

报告的标题固定格式：`{基金名称} — {案由}研究简报`

---

## 完整模板（买入案由）

报告最前面展示的应该是结论，第一行用户就能读到。

```
结论：{买 / 不买 / 持有 / 卖出 / 等信号}
可信度：{X}/10
预期 1 年收益：{区间，如 +5%~+15%}
一句话：{为什么是这个结论}
如果只能记住一件事：{the one thing}
翻案条件：{什么新信息出现时本结论应被推翻}

主要风险：
1. {风险描述} — 概率：{低/中/高}
2. {风险描述} — 概率：{低/中/高}

关键证据：
1. {证据} — 置信度: 高/中/低 — 来源: {来源名}
2. {证据} — 置信度: 高/中/低 — 来源: {来源名}
3. {证据} — 置信度: 高/中/低 — 来源: {来源名}

买入的前提条件（如果结论是"等信号"，写清楚等什么）：

退出信号（什么触发时无条件卖出）：

不确定项：
- {不确定项} — 值得跟进 ✓ / 可以接受 ✗

来源：
- {来源名}（可信度：高/中/低）
- {来源名}（可信度：高/中/低）
```
