# 证据分析规则

每个证据维度的指标库和解读方法。按需查阅，不全量加载。

---

## 1. 业绩归因

**核心追问**：收益来自运气还是能力？alpha 还是 beta？

### 指标
| 指标 | 计算/获取 | 解读 |
|------|-----------|------|
| 累计收益率 | 来源：天天/晨星 | 看时间匹配，不跨期比较 |
| 年化收益率 | 来源：晨星/Choice | > 同类均值 → 正超额 |
| 超额收益（alpha） | 收益 - 基准收益 | 正超额才有讨论价值的必要 |
| 信息比率 | 超额收益 / 跟踪误差 | > 0.5 一般，> 1 优秀 |
| 滚动收益 | 3 月/6 月/1 年滚动 | 稳定性 vs 爆发力：连续为正还是偶尔冲刺 |
| 同类排名 | 四分位 / 百分位 | 近 1/3/5 年排名趋势比单年排名重要 |

### 因果链追问
- 超额收益来自行业配置还是个股选择？（Brinson 归因思路）
- 超额收益是否集中在某一年？（运气信号）
- 排名趋势是上升还是下降？（能力退化/经理更换）

## 2. 持仓分析

**核心追问**：基金赌了什么？集中度风险多大？

### 指标
| 指标 | 解读 |
|------|------|
| 前 10 持仓占比 | > 60% 高集中度，涨跌同向风险大 |
| 行业分布（前 3 行业占比） | > 50% 行业集中，板块轮动中可能大起大落 |
| 持股数量 | < 20 只极集中，> 100 只高度分散 |
| 换手率 | > 200% 高频交易，可能是交易型风格也可能是过度交易 |
| 港股/美股持仓占比 | 跨市场配置比例 |
| 重仓股变动 | 季度间前 10 变化 > 5 只 → 风格可能漂移 |

### 因果链追问
- 高集中度是主动选择还是被动结果？（经理风格 vs 规模限制）
- 换手率与收益正相关吗？（高换手是否产生超额）
- 前 10 持仓是长期持有还是季度轮换？

## 3. 经理评估

**核心追问**：这个人可靠吗？他的决策可预测吗？

### 指标
| 指标 | 解读 |
|------|------|
| 任职年限 | < 1 年 → 观察期；1-3 年 → 初步可信；> 5 年 → 稳定期 |
| 任职以来回报 vs 前任/同类 | 换经理后业绩变化方向 |
| 经理同时管理基金数 | > 5 只 → 精力分散风险 |
| 卸任历史 | 频繁卸任/跳槽 → 稳定性存疑 |
| 从业背景 | 研究员出身 vs 交易员出身 vs 量化背景 |
| 公开言论 vs 实际持仓 | 言行一致度——说看好消费但持仓里没有 → 风格漂移预警 |
| 自购份额 | 经理自购金额和趋势——真金白银是最高置信度信号 |

### 因果链追问
- 业绩好是因为经理能力强还是市场β？
- 经理跳槽对基金的影响？新经理风格和前任一致吗？
- 经理是否在管规模快速膨胀的产品？规模是业绩的敌人吗？

## 4. 风格漂移

**核心追问**：说好的价值型，怎么在买成长？

### 指标
| 指标 | 解读 |
|------|------|
 | 持仓风格箱 | 晨星九宫格（价值/平衡/成长 × 大盘/中盘/小盘）的变化轨迹 |
| 行业配置偏移 vs 基准 | 连续偏离 → 可能正在漂移 |
| 市值分布变化 | 从大盘漂到小盘 → 风格位移 |
| 招股说明书风格承诺 vs 实际 | 合同约定 vs 实际行动的差距 |
| 名称与行为一致性 | 基金名称含「成长」但持仓 PE 低于同类均值 |

### 因果链追问
- 漂移是主动战术调整还是风格框架换了？
- 漂移后业绩变好还是变差？（漂移未必错，但需要确认是有意为之还是纪律松懈）
- 如果是主动调仓，是否伴随公开说明？

## 5. 风险矩阵

**核心追问**：极端情况下会亏多少？

### 指标
| 指标 | 解读 |
|------|------|
| 最大回撤 | 历史最大亏损幅度。> 同类均值 2x → 非对称下行风险 |
| 回撤恢复时间 | 几次分红后净值修复速度 |
| 年化波动率 | > 同类均值 → 高风险特征 |
| 下行风险（Downside Deviation） | 只关心亏损波动，比波动率更准确 |
| VaR (95%) | 一天/一月内最坏情况估计 |
| 与基准相关性 | > 0.9 → 基本是 β，alpha 空间小 |
| 正收益月占比 | 月胜率 > 60% 稳定性较好 |

### 因果链追问
- 最大回撤发生在什么市场环境下？之后如何修复？
- 回撤原因是市场整体下行还是基金自身问题？
- 风险指标是否在恶化（经理风险控制能力下降）？

## 6. 费率与效率

**核心追问**：管理费值不值？

### 指标
| 指标 | 解读 |
|------|------|
| 管理费率 | 主动管理 > 1.5% 算高，ETF < 0.5% 正常 |
| 托管费率 | 通常 0.1%-0.25% |
| 申赎费率 | 持有期越短赎回费越高。注意短期交易成本 |
| 净收益率（扣除费用后） | 这才是投资者实际到手的 |
| 管理费 vs 超额收益 | 如果费后超额收益 < 0 → 费率太贵 |
| 规模 vs 费率 | 大规模基金费率通常有议价空间 |

### 因果链追问
- 费率在同类中处于什么水平？
- 费率和业绩匹配吗？（高费率基金是否真的产生了高 alpha？）
- 规模增长后费率是否下调？

## 7. 同类对比

**核心追问**：在同一赛场里排名如何？

### 指标
| 指标 | 解读 |
|------|------|
| 四分位排名 | 近 1/3/5 年分别看，关注趋势而非单年 |
| 同类平均 vs 基金 | 超额的方向和幅度 |
| 排名稳定性 | 连续 3 年在前 50% → 稳定型选手 |
| 同类比较时匹配的投资范围 | 偏股混 vs 偏债混 vs 灵活配置——不同类别不直接比 |
| 规模排名 | 同类规模分位 → 规模是否大到影响灵活性 |

### 因果链追问
- 排名上升是什么原因？（市场风格刚好匹配？还是经理能力提升？）
- 同类中的「幸存者偏差」——分母是否包含了已清盘的基金？
- 排名靠前是否因为承担了更高风险？风险调整后排名如何？

## 8. 基金公司

**核心追问**：背后的投研体系可靠吗？

### 指标
| 指标 | 解读 |
|------|------|
| 公司管理规模 | 头部公司资源更充沛，但灵活性可能下降 |
| 投研团队人数/稳定性 | 频繁离职 → 投研文化问题 |
| 旗下同类基金表现一致性 | 如果同公司同类基金表现差异极大 → 体系支撑弱，更依赖个人 |
| 合规记录 | 有没有违规被处罚的历史 |
| 股权结构 | 券商系 / 银行系 / 个人系 — 影响激励机制和投资文化 |

### 因果链追问
- 公司投研文化是「明星经理制」还是「平台制」？
  - 明星制 → 经理离职影响大
  - 平台制 → 可持续性更强
- 公司规模增长后，是否出现规模不经济？

## 9. 资金流向

**核心追问**：聪明钱在进还是出？

### 指标
| 指标 | 解读 |
|------|------|
 | 规模变化趋势 | 连续 2 季度规模下降 → 可能有问题 |
| 机构持有比例变化 | 机构投资者比例上升/下降——机构通常比散户更有信息优势 |
| 大额赎回 | 单日大额赎回 → 可能是机构撤离 |
| 分红频率 | 频繁分红 → 控制规模？还是税务优化？ |
| 限购/开放状态 | 限购 → 经理认为规模已经影响业绩 |
| 申购 vs 赎回趋势 | 净申购→信心；净赎回→不信任 |

### 因果链追问
- 机构资金进出是否与业绩趋势一致？（不一致时，机构可能掌握了未公开信息）
- 规模快速膨胀是否发生在业绩高点附近？（追涨效应）
- 限购是主动风控还是被动的持有人结构问题？

## 10. 前瞻与情景

**核心追问**：未来会怎样？什么会推翻这个结论？

### 市场映射（当前环境）
- 当前市场阶段：牛 / 熊 / 震荡 / 风格切换期
- 基金历史表现与市场阶段的对应关系
- 基金持仓与当前市场风格的相关性
- 宏观因子敏感度：利率 / 流动性 / 经济周期

### 催化剂扫描（未来 6 个月）
- 持仓股未来 3 个月有财报/重大事件的占比
- 相关行业政策窗口期（集采/补贴/监管新规等）
- 基金重大事项预告（分红/限购变更/经理变动传闻）
- 市场整体风险偏好变化趋势

### 反方情景
- 如果结论是买入：最可能因为什么错？概率多大？
- 如果结论是卖出：最可能踏空什么？概率多大？
- 历史上类似情景下，基金表现如何？
- 反方情景是尾部风险还是中等概率事件？

### 指标
| 指标 | 解读 |
|------|------|
| 基金 vs 市场风格匹配度 | 如果市场偏好价值但基金重仓成长 → 逆风 |
| 历史情景回测 | 基金在历史上类似市场环境的表现 |
| 行业轮动阶段 | 基金重仓行业处于轮动周期的什么位置 |
| 利率敏感度 | 债基/偏债混特别关注 |
| 宏观经济因子暴露 | 基金对通胀/利率/GDP 的敏感度 |
| 当前排名 vs 风格匹配度 | 排名高但风格不匹配 → 确认偏误风险；排名低且不匹配 → 可能等到风格回归 |
| 催化剂密度 | 未来 3-6 月持仓股有财报/政策事件的占比 → 高密度意味着潜在波动 |
| 反方情景概率 | 看错的情况下，最可能的原因和概率 → 辅助置信度校准 |

### 因果链追问
- 当前的业绩表现是风格匹配的结果还是真实 alpha？
- 如果风格转换，基金能否适应？（经理的历史调仓能力）
- 当前市场环境下最好的策略是什么？这个基金在践行这个策略吗？
- 催化剂是否已被市场定价？如果尚未反映，这个基金是受益方还是受损方？
- 反方情景发生概率是否已体现在置信度中？还是确认偏误让你低估了它？
