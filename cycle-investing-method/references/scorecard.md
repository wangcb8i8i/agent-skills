# 行业周期投资 · 基金评分卡

用行业周期投资框架（见 `method.md`）评价**任意基金**。分数衡量**该基金对周期投资框架的契合度**，不是基金质量或未来表现。

**重要**：一只基金可以很优秀但评分低（如纯债基金、低波红利基金、买入持有型核心基金）。低分 ≠ 差基金——只说明它遵循不同的投资理念。

评分流程：先跑 `scripts/score_fund.py <基金代码>` 备好机械指标，再按下面六个维度逐项打分。

---

## 六维评分（总分 100）

### 1. 景气方向与通胀属性（25 分）

**看什么**：前十大是否集中在当前景气上行、有定价权/涨价的环节？通胀是科技驱动的（供给端创造需求）还是传统供需错配？

**方法依据**：§1 —— 投资行业周期拐点，偏爱科技驱动的通胀。

**打分**：
| 20–25 | 明确押注在景气上行周期中的科技通胀方向（AI 基础设施、光通信、半导体、储能等） |
| 10–19 | 部分沾边 — 周期暴露与其他风格混合 |
| 0–9 | 以防御/红利/固收/低景气周期为主 |

### 2. 低 ROE 修复弹性（20 分）

**看什么**：组合是否偏好当前 ROE 较低但有可信修复路径的公司（供求压制型或预付研发型）？还是清一色高 ROE 白马？

**方法依据**：§4b — 偏好低 ROE 修复弹性而非高 ROE 龙头。

**打分**：
| 15–20 | 明显倾向低 ROE 修复标的 / 中小盘成长 |
| 8–14 | 混合 — 部分低 ROE，部分成熟型 |
| 0–7 | 以高 ROE 蓝筹 / 红利股为主 |

*注：个股精确 ROE 需要验证；可用中小市值占比和行业定位作为代理指标。*

### 3. 全球视野与比较优势（15 分）

**看什么**：基金定位是否符合"全球周期 + 本地比较优势"的逻辑链，还是纯国内/主题炒作？

**方法依据**：§3 — 从全球拐点出发，落到本地优势。

**打分**：
| 11–15 | 清晰的全球周期→本地优势链条（如光通信、电池供应链、半导体设备） |
| 5–10 | 部分 — 有些全球逻辑，混杂国内主题 |
| 0–4 | 纯内需/本地主题，与全球周期无关 |

### 4. 流动性（10 分）

**看什么**：持仓是否有足够流动性在需要时退出？基金规模与持仓风格是否匹配？

**方法依据**：§4a — 流动性第一。

**打分**：
| 7–10 | 多为大中盘流动性好的标的，基金规模适配 |
| 4–6 | 混合 |
| 0–3 | 大量小盘低流动性的标的，或基金规模对持仓风格来说太大 |

*注：个股流动性细节需核实。*

### 5. 集中度与周期拼接（15 分）

**看什么**：基金是**集中在周期主题上且积极调仓**（高换手、周期拼接），还是（a）集中但不动，或（b）分散但不动？

**方法依据**：§6 — 周期拼接、逐步拟合，高换手是特征。

**打分**：
| 11–15 | 集中在周期主题 + 可见的季度间调仓（周期拼接） |
| 5–10 | 集中但不动，或分散无明确周期主线 |
| 0–4 | 既分散又不动 |

*注：相邻季度前十大重叠率可代理换手。高换手在此框架中是加分项。*

### 6. 业绩与回撤印证（15 分）

**看什么**：收益是否来自抓住了正确的周期上行？回撤特征是否符合高波动周期投资的特点（高上行，回撤显著但不致命）？

**方法依据**：§5 — 回撤靠选对周期控制，非对冲；§7 — 重过程、轻结果。

**打分**：
| 11–15 | 在周期上行期大幅跑赢基准，收益明显与周期定位相关 |
| 5–10 | 表现中等，收益来源混杂 |
| 0–4 | 持续跑输，或收益来自非周期来源（如票息、分红） |

## 输出格式

```
# 【基金名（代码）】：XX / 100

一句话结论。

| 维度 | 得分 | 证据（持仓/数据，带季度） |
|------|------|------------------------|
| 1 景气方向 | x/25 | ... |
| 2 低 ROE 弹性 | x/20 | ...（个股ROE需核实） |
| 3 全球优势 | x/15 | ... |
| 4 流动性 | x/10 | ... |
| 5 周期拼接 | x/15 | 前十大集中度 X%、季度换手代理 Y |
| 6 业绩印证 | x/15 | 近1年/成立以来/最大回撤 ... |

总分 XX/100
评级：高度契合（80+）/ 较契合（60-79）/ 部分契合（40-59）/ 不契合（<40）

核心优势与短板（2-3句）。

本评分衡量的是对行业周期投资框架的契合度，非基金绝对质量。"需核实"项请自行验证。
```

## 行业周期阶段参考

评分维度 1（景气方向）时，可将持仓映射到当前周期阶段：

| 行业 | 当前阶段（示意，需更新） | 类型 |
|------|------------------------|------|
| AI 基础设施 / 光通信 | 上行 / 通胀 | 科技驱动 |
| 半导体（先进制程） | 上行 | 科技驱动 |
| 储能 / 电动车供应链 | 成熟 / 周期恢复 | 科技驱动→传统 |
| 大宗商品（铜、油） | 视周期而定 | 传统 |
| 消费 / 互联网 | 成熟 / 低通胀 | 成熟型 |
| 防御 / 公用事业 | 稳定 / 低通胀 | 防御型 |
| 固收 / 货币 | 不适用 | 不适用 |

*周期阶段会变化，定期更新。*
